# Vasanth M Portfolio

Clean static portfolio for Vasanth M, Java Backend Developer.

## Files

- `index.html` - main portfolio page
- `styles.css` - responsive styling
- `script.js` - mobile menu and copy-email behavior
- `assets/Vasanth_M_Resume.pdf` - resume PDF
- `robots.txt` and `sitemap.xml` - basic SEO files

## Deploy

Upload all files to your hosting provider's public folder. The site does not need a build step.

## Edit Before Uploading

- Replace the GitHub profile link with the exact URL Shortener repository if available.
- Update the resume PDF when your resume changes.
